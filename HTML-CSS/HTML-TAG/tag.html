Step by Step Tutorial: Basic HTML Lists

This is a complete example using only HTML tags. No CSS is used anywhere.

Copy the entire code below into a file named lists-tutorial.html and open it in any browser.

Step 1: What are HTML Lists?

HTML has two basic list types:

  • <ul> Unordered List → shows bullets
  • <ol> Ordered List → shows numbers
  • <li> List Item → used inside both

Step 2: How to write an Unordered List

Basic code pattern:

<ul>
  <li>Item One</li>
  <li>Item Two</li>
  <li>Item Three</li>
</ul>

Result looks like this:

  • Item One
  • Item Two
  • Item Three

Step 3: How to write an Ordered List (Step-by-Step Example)

Basic code pattern:

<ol>
  <li>First step</li>
  <li>Second step</li>
</ol>

Here is a real step-by-step tutorial using an ordered list:

  1. Open any text editor (Notepad, VS Code, etc.)
  2. Create a new file and save it as mylist.html
  3. Write <!DOCTYPE html> at the top
  4. Add <html>, <head>, and <body> tags
  5. Inside <body> add your <ol> or <ul> list
  6. Save the file again
  7. Double-click the file to open it in your browser
  8. Congratulations! You created your first HTML list

Step 4: Nested Lists (Lists inside lists)

You can put one list inside another list item:

  • Fruits
    • Apple
    • Banana
    • Mango
  • Vegetables
    • Carrot
    • Tomato

Step 5: Full Basic HTML Template (Copy & Use)

Here is the complete ready-to-use code you can copy:

<!DOCTYPE html>
<html>
<head>
    <title>My List Page</title>
</head>
<body>
    <h1>My Shopping List</h1>
    
    <h2>Things to Buy</h2>
    <ul>
        <li>Milk</li>
        <li>Bread</li>
        <li>Eggs</li>
    </ul>

    <h2>Steps to Make Tea</h2>
    <ol>
        <li>Boil water</li>
        <li>Add tea leaves</li>
        <li>Add sugar and milk</li>
        <li>Stir and serve</li>
    </ol>
</body>
</html>

You did it!

You have now learned how to create basic lists in HTML using only HTML tags.

Practice by changing the items in the lists above and save the file again.
